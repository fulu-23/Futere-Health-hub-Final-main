## Website desciption
Future health Hub is a website created to help learners understand different carrers in the healthfield.many young people are
not sure what each career involves,so this platform gives simple and clear information to guide them. 



## Features
- easy to use and understand
- simple navigation between pages
- information about different health careers
- contact and enquiry options
- improve styling and user experiences
