## Project Overview
**Future Health Hub** is a responsive web platform designed to help learners explore and understand careers in the healthcare field. Many young people are unsure about what different health careers involve—so this website provides clear, simple information to guide decision-making. The site includes basic career guidance content, plus contact and enquiry features to support users who want to ask questions.

---

## Website Goals and Objectives
### Goals
1. Provide accessible career guidance for learners interested in healthcare careers.
2. Improve user experience with clear navigation and modern styling.
3. Encourage user interaction through contact and enquiry options.

### Objectives
1. Explain key aspects of selected healthcare careers in an easy-to-read format.
2. Ensure the website is fully responsive across mobile, tablet, and desktop.
3. Provide simple and consistent navigation across all pages.
4. Allow users to submit enquiries and contact information.

## Key Features and Functionality
### Pages / Content Sections
- **Home (Index):** Hero message introducing the platform and quick highlights.
- **About Us:** Background information about the project and purpose.
- **Services / Careers:** Career-focused content presented in sections/cards.
- **Contact:** Contact details and/or contact form section.
- **Enquiries:** Enquiry information and enquiry form section.

### Functionality (UI/UX)
- **Simple navigation** between pages via a consistent header menu.
- **Responsive design** so content adapts to smaller screens.
- **Card-based layout** and clear typography to improve readability.
- **Improved visual styling** (colour palette, spacing, layout hierarchy).
- **Client-side JavaScript support** (where used) to enhance interactivity/experience.

### Technologies Used
- **HTML5** — page structure and semantic content
- **CSS3** — responsive styling, layout, colours, typography
- **JavaScript** — interactive enhancements (where applicable)
- **GitHub Pages** — hosting/deployment
