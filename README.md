## 📖 About the Project

Many young people reach the end of school without a clear picture of what a career in healthcare actually involves. **Future Health Hub** was built to close that gap — it presents clear, beginner-friendly information about a range of health careers, helping learners make more informed decisions about their future.

The project was developed as a coursework submission for the **WEDE5020 Web Development** module, with a strong focus on usability, visual consistency, and responsive design.

## 📁 Code Quality

All HTML, CSS, and JavaScript files include **descriptive inline comments** explaining key structural sections — including navigation, the hero section, the footer, and form styling.

The Contact and Enquiry forms are backed by a dedicated JavaScript module that handles:

- ✅ Field validation (name, email, phone, and message format checks)
- ⏱️ Basic rate limiting to prevent repeated rapid submissions
- 🚫 Lightweight spam and malicious-input filtering
- 💬 Clear, accessible success and error feedback for users

## 🚀 Getting Started

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/fulu-23/Futere-Health-hub-Final-main.git

# 2. Navigate into the project folder
cd Futere-Health-hub-Final-main

# 3. Open index.html in your browser
```

No build tools or installations required — it's built with plain HTML, CSS, and JavaScript, ready to view in any modern browser (Chrome, Firefox, Edge, or Safari).
